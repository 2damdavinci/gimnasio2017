Clases Grupales

La tabla cargara automaticamente todas las clases grupales.
Para añadir una nueva clase, pulse el botón Añadir y rellene el formulario
Para modificar una clase, seleccionela de la lista, pulse el boton Modificar y rellene el formulario
Para eliminar una clase, seleccionela de la lista y pulse el boton Eliminar.
Para obtener la lista de horarios de una clase, haga doble clic sobre ella.
Para obtener un informe con la lista de horarios de una clase, selecciónela de la lista y pulse el botón Generar Informe.

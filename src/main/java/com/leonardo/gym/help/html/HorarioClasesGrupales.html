Horarios Clases Grupales

Para cargar los horarios de una clase, haga doble clic sobre ella.
Para añadir unn nuevo horario, pulse el botón Añadir y rellene el formulario
Para modificar un horario, selecciónelo de la lista, pulse el botón Modificar y rellene el formulario
Para eliminar un horario, seleccióonelo de la lista y pulse el botón Eliminar.
Para obtener la lista de alumnos y detalles de un horario, haga doble clic sobre él. Para obtener un informe con la lista de alumnos y detalles de una clase, selecciónela de la lista y pulse el botón Generar Informe.
